# WorkOS PHP SDK

## SDK Generation

This SDK uses code generation from OpenAPI specifications. When generating or modifying SDK code, follow the design patterns in `.claude/SDK_DESIGN.md`.

## Available Skills

- `/generate-sdk <spec_path>` - Generate complete SDK from OpenAPI
- `/generate-sdk-models <spec_path>` - Generate models only
- `/generate-sdk-resources <spec_path>` - Generate resources only
- `/generate-sdk-tests <spec_path>` - Generate tests only
